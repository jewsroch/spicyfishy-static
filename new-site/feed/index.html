Yes, this is WordPress.. ( I added a link at the bottom of the theme so it’s more apparent.). I haven’t seen a whole lot about typepad to wordpress imports, but then again, I haven’t really looked A quick google search came up with a bunch of hits, so it looks pretty possible.

As for our transition – the blogger to wordpress was pretty simple. There is a built-in wordpress importer that can bring over your posts, comments, permalinks etc from blogger or typepad (and a bunch of other services). It didn’t import my images, which is annoying because Blogger will only allow me to link to them for a short time. There are some plugins that move them over to my host, but they’re really old and don’t work. Hopefully someone will write a new one soon. The setting up part was a bit more involved, mainly because of the photo hosting. There isn’t an amazing built in photo upload/hosting solution in wordpress. It has a media manager but it’s a bit wonky. That said, wordpress is great because of all the plugins and addons you can put in it to do whatever you want it to. So adding in a few plugins and other photo hosting database thingies made it happen ok. I’m still not perfectly pleased with the way we have the photo stuff setup, but it’s a good start. More plugins will be written, or I’ll find something else, and all will be well. Hopefully.

You should look into Moveable Type too. I think it’s more closely related to typepad and it’s supposed to be a little more user friendly start and go. Less plugins etc as it’s not a freeware/opensource kinda thing, but supposed to be very cool too.

That’s my two sense. I’m enjoying wordpress and it is nice to be able to have static pages (like for your site and our photo pages).
Good luck,
Thanks for the Congrats.. we’re pretty excited
